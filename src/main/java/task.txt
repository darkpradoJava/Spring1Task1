Создайте POJO-class Cat
в кассе AppConfig по аналогии создайте бин с именем “cat”.
Настройте этот бин так, чтобы он создавался новым при каждом запросе.
В классе App по аналогии вызовите еще раз бин HelloWorld, затем 2 раза вызовите бин cat.
Сравните 2 пары бинов по ссылке и выведите результат в консоль.
Для пары хэлловорлдов должно вернуться тру, для кэт - фоллс.
